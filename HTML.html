What is it?

HTML is a standard markup language for documents to be displayed in web browser. It serves as the layout of the document's content in a web browser by using what's called Document Object Model(DOM) Tree

History

1991-1993: Basic Creation (HTML 1.0)

Tim Berners Lee created the HTML in the late 1991, and released HTML 1.0 in 1993. This is created to share information that is readable and accessible via web browsers

1995-1997: Official Release of HTML (HTML 2.0)

Official publication of HTML as HTML 2.0 which contains all the features of HTML 1.0 along with additional features. HTML became the official standard markup language for designing and creating websites, and refined various core features of HTML

1997: (HTML 3.0)

HTML 3.0 was drafted by Dave Ragget which included new features of HTML, introducing more powerful features and characteristics for designing webpage, but these features also slowed down the browser
